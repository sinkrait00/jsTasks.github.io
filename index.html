Задача №1

Задание №1

for (var i = 0; i < 10; i++) {
setTimeout(function() {
console.log(i);
}
};

Что выведет консоль и почему?

Ответ

Выведется число 10, т.к. setTimeout начинает свою работу только после завершения цикла и используется переменная i, которая уже равна 10
